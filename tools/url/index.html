FAQ

What are URL Encode and URL Decode?

URL Encode converts spaces and special characters into a safe format for URLs. URL Decode converts encoded text back into a readable form.

When should I use full URL mode?

Use full URL mode when the input is an entire link. It keeps structural characters such as http://, ?, &, and = while encoding unsafe characters.

When should I use URL component mode?

Use URL component mode for parameter values such as name, keyword, redirect_uri, or callback_url.
