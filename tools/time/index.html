FAQ

What is the difference between 10-digit and 13-digit timestamps?

10 digits usually means seconds. 13 digits usually means milliseconds.

Does this tool upload my data?

No. The tool runs in your browser. Local history is stored only in browser localStorage.

What can this timestamp tool be used for?

It is useful for Unix timestamp conversion, date to timestamp, UTC to local time, time zone conversion, batch conversion, and log time debugging.
