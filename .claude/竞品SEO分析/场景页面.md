# InfiniSynapse 场景页面规划

> 生成日期：2026-05-28
> 方法论：从产品文档提取 InfiniSynapse 核心能力 → 匹配有差异化的行业/场景 → 从竞品已有 SEO 页面中选取真实关键词（不编造）

---

## 一、筛选逻辑

从产品文档中提取 InfiniSynapse 独有的、竞品难以复制的 5 个能力：

| 能力 | 为什么竞品难复制 |
|---|---|
| **多源联合分析** | Julius/Powerdrill 是 file-first，Databricks/Snowflake 锁自己生态 |
| **知识库 + 联网搜索双通道** | 竞品最多做其一，没有同时做的 |
| **Browser Use 网页数据采集** | 只有 Exa/You.com 做搜索 API，不做浏览器操作 |
| **结构化 + 非结构化联合分析** | 竞品要么只做数据库要么只做文档 |
| **企业私有部署（Air-Gapped）** | Julius/Powerdrill/Hex 全是 SaaS-only |

**筛选原则：** 只选 InfiniSynapse 能力有差异化优势、且竞品已有 SEO 关键词的行业/场景。
