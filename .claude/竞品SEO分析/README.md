## 分析模板（每次分析统一使用）

每份竞品 SEO 分析报告包含以下模块：

1. **站点基础信息** — URL、Title、Meta、域名、托管、产品定位、定价
2. **博客内容策略** — 文章列表、目标关键词、内容类型分类
3. **关键词体系分层** — 品牌词 → 品类词 → 场景词 → 竞品替代词 → 技术热词
4. **SEO 落地页结构模式** — 通用页面结构拆解
5. **第三方定位** — 外部评测网站对该竞品的评价
6. **对 InfiniSynapse 的启示** — 可复用策略 + 竞品未覆盖的空白
7. **数据来源** — 所有信息的出处
